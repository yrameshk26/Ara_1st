# Background Music

Place your birthday music file here with the name:
- **birthday-music.mp3**

**Recommended specifications:**
- Format: MP3 (for best browser compatibility)
- Length: 2-5 minutes (it will loop automatically)
- File size: Keep under 5MB for faster loading
- Volume: Normalized/balanced audio (the website sets it to 30% by default)

**Suggested music types:**
- Instrumental birthday music
- Soft background melodies
- Celebratory but not overwhelming
- Royalty-free music to avoid copyright issues

**Where to find royalty-free music:**
- YouTube Audio Library
- Free Music Archive
- Incompetech
- Bensound
- Purple Planet Music

**Note:** The music will only start playing after the user clicks "Start Experience" button, complying with browser autoplay policies.
